基本数据类型

  • 自动类型转换：
    精度小的类型自动转换为精度大的数据类型，数据精度类型按精度大小排序为
    • char → int →long → float → double
    • byte → short → int → long → float → double

基本运算

int i = 1;
i = i++;
System.out.printIn(i);//输出：1，原因：i++是先赋值后加一；++i是先加一后赋值
